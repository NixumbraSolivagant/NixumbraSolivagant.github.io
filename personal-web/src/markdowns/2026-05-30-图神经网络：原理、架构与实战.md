# 图神经网络：原理、架构与实战

> *"现实世界中的大多数数据都可以建模为图结构，而图神经网络正是处理这类非欧几里得数据的革命性工具。"*
> — Michael Bronstein，图神经网络领域先驱

## 什么是图神经网络？

**图神经网络**（Graph Neural Network，GNN）是一类专门用于处理**图结构数据**的深度学习模型。与传统神经网络擅长处理规则的网格数据（如图像、文本）不同，GNN 能够有效处理具有不规则拓扑结构的数据，将深度学习的强大能力带入了关系数据的领域。

### 为什么需要图神经网络？

在现实世界中，图结构数据无处不在：

| 应用领域 | 图结构 | 节点/边含义 |
|----------|--------|------------|
| 社交网络 | 用户社交图 | 节点=用户，边=好友关系 |
| 分子化学 | 分子图 | 节点=原子，边=化学键 |
| 推荐系统 | 用户-物品交互图 | 节点=用户/商品，边=交互 |
| 知识图谱 | 实体关系图 | 节点=实体，边=关系 |
| 交通网络 | 道路网络 | 节点=路口，边=道路 |
| 交通网络 | 道路网络 | 节点=路口，边=道路 |

传统深度学习方法难以直接处理这类数据，原因在于：

1. **非欧几里得结构**：图像有固定的空间位置，文本有线性序列，但图没有固定的节点顺序
2. **可变的拓扑结构**：不同图的节点数和边数可能完全不同
3. **节点间复杂的依赖关系**：需要建模多跳邻居的影响

## 图的基本概念

在深入 GNN 之前，我们需要明确图论中的核心概念。

### 图的数学定义

一个图 $G$ 可以表示为 $G = (V, E)$，其中：

- $V = \{v_1, v_2, \ldots, v_N\}$：节点集合，$|V| = N$ 为节点数
- $E \subseteq V \times V$：边集合，$|E| = M$ 为边数

### 邻接矩阵

邻接矩阵 $A \in \mathbb{R}^{N \times N}$ 描述节点间的连接关系：

$$A_{ij} = \begin{cases} 1 & \text{如果节点 } i \text{ 和 } j \text{ 之间有边} \\ 0 & \text{否则} \end{cases}$$

对于**无向图**，$A$ 是对称矩阵；对于**有向图**，$A$ 不一定对称。

### 度矩阵

节点的**度**（degree）表示与其相连的边的数量：

$$d_i = \sum_{j=1}^{N} A_{ij}$$

度矩阵 $D$ 是一个对角矩阵：$D_{ii} = d_i$

### 拉普拉斯矩阵

拉普拉斯矩阵是图论中的核心工具：

$$L = D - A$$

标准化形式（对称归一化）：

$$L_{sym} = D^{-\frac{1}{2}} L D^{-\frac{1}{2}} = I - D^{-\frac{1}{2}} A D^{-\frac{1}{2}}$$

拉普拉斯矩阵的特征值和特征向量揭示了图的结构性质，在谱图理论中至关重要。

### 图的分类

| 类型 | 定义 | 特点 |
|------|------|------|
| 有向图 | 边有方向 | $A_{ij} \neq A_{ji}$ |
| 无向图 | 边无方向 | $A$ 对称 |
| 加权图 | 边有权重 | $A_{ij} = w_{ij}$ |
| 有属性图 | 节点/边有特征 | $X_v, X_{uv}$ |
| 多重图 | 节点间多条边 | $A$ 为整数 |
| 超图 | 一条边连接多个节点 | 需要特殊处理 |

## 图卷积网络 (GCN)

**图卷积网络**（Graph Convolutional Network，GCN）是最基础且应用最广泛的图神经网络，由 Thomas Kipf 和 Max Welling 于 2017 年提出。

### 核心思想

GCN 的核心思想是**节点的表示由其自身特征和邻居节点特征共同决定**。通过聚合邻居信息，节点能够"看到"图中的局部结构。

### 层间传播规则

GCN 的层间传播公式为：

$$H^{(l+1)} = \sigma\left(\tilde{D}^{-\frac{1}{2}} \tilde{A} \tilde{D}^{-\frac{1}{2}} H^{(l)} W^{(l)}\right)$$

其中：
- $\tilde{A} = A + I$：加入自环的邻接矩阵
- $\tilde{D}$：$\tilde{A}$ 的度矩阵
- $H^{(l)}$：第 $l$ 层的节点表示矩阵
- $W^{(l)}$：第 $l$ 层的可学习权重矩阵
- $\sigma$：非线性激活函数（通常是 ReLU）

### 物理意义解读

这个公式可以从两个角度理解：

**1. 空间域理解**：每个节点从邻居节点收集信息，并与自己的特征加权融合

$$h_i^{(l+1)} = \sigma\left(\sum_{j \in \mathcal{N}(i) \cup \{i\}} \frac{1}{c_{ij}} W^{(l)} h_j^{(l)}\right)$$

其中 $c_{ij} = \sqrt{d_i d_j}$ 是归一化系数。

**2. 谱域理解**：卷积操作在图的拉普拉斯矩阵的谱域中进行，等价于对图信号进行低通滤波。

### 代码实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class GraphConvolution(nn.Module):
    """图卷积层实现"""
    def __init__(self, in_features, out_features, bias=True):
        super(GraphConvolution, self).__init__()
        self.in_features = in_features
        self.out_features = out_features
        self.weight = nn.Parameter(torch.FloatTensor(in_features, out_features))
        if bias:
            self.bias = nn.Parameter(torch.FloatTensor(out_features))
        else:
            self.register_parameter('bias', None)
        self.reset_parameters()

    def reset_parameters(self):
        nn.init.xavier_uniform_(self.weight)
        if self.bias is not None:
            nn.init.zeros_(self.bias)

    def forward(self, input, adj):
        # input: 节点特征 [N, in_features]
        # adj: 邻接矩阵 [N, N]
        support = torch.mm(input, self.weight)  # 线性变换
        output = torch.spmm(adj, support)       # 稀疏矩阵乘法聚合
        if self.bias is not None:
            output = output + self.bias
        return output

class GCN(nn.Module):
    """两层图卷积网络"""
    def __init__(self, n_features, n_hidden, n_classes, dropout=0.5):
        super(GCN, self).__init__()
        self.gc1 = GraphConvolution(n_features, n_hidden)
        self.gc2 = GraphConvolution(n_hidden, n_classes)
        self.dropout = dropout

    def forward(self, x, adj):
        x = F.relu(self.gc1(x, adj))
        x = F.dropout(x, self.dropout, training=self.training)
        x = self.gc2(x, adj)
        return F.log_softmax(x, dim=1)
```

### GCN 的局限性与改进方向

| 局限性 | 说明 | 改进方案 |
|--------|------|----------|
| **感受野固定** | 只看一阶邻居 | 堆叠多层 GCN |
| **缺乏灵活性** | 聚合方式单一 | 引入注意力机制 (GAT) |
| **无法处理异构图** | 所有节点类型相同 | R-GCN, HAN |
| **transductive 学习** | 需要所有节点参与训练 | GraphSAGE 归纳学习 |
| **过平滑问题** | 层数加深导致节点表示趋同 | Jump Knowledge Network |

## GraphSAGE：归纳式学习

**GraphSAGE**（Graph SAMPLE and AGgregatE）由 William Hamilton 等人于 2017 年提出，解决了 GCN 无法泛化到未见节点的问题。

### 核心思想

GraphSAGE 的创新在于**学习一个聚合函数**，而不是为每个节点学习独立的嵌入。这个聚合函数可以从节点的局部邻域中归纳生成节点嵌入。

### 聚合过程

GraphSAGE 的前向传播包含三个关键步骤：

**1. 邻居采样**：从每个节点的邻居中随机采样固定数量的节点

```python
def sample_neighbors(self, nodes, num_samples):
    """采样固定数量的邻居"""
    sampled = []
    for node in nodes:
        neighbors = self.adj_list[node]
        if len(neighbors) > num_samples:
            sampled.append(random.sample(neighbors, num_samples))
        else:
            sampled.append(neighbors)
    return sampled
```

**2. 邻居聚合**：将采样得到的邻居特征聚合成一个向量

三种经典聚合器：

```python
class MeanAggregator(nn.Module):
    """均值聚合器"""
    def forward(self, features, neighbors):
        # neighbors: 邻居节点特征列表
        # 均值聚合
        neighbor_hidden = torch.stack([features[neighbor] for neighbor in neighbors])
        return torch.mean(neighbor_hidden, dim=0)

class LSTMAggregator(nn.Module):
    """LSTM聚合器 - 考虑邻居顺序"""
    def forward(self, features, neighbors):
        # 对邻居进行随机排序（打破对称性）
        shuffled = random.shuffle(neighbors)
        # 使用LSTM处理序列
        lstm_input = torch.stack([features[neighbor] for neighbor in shuffled])
        lstm = nn.LSTM(self.embedding_dim, self.embedding_dim)
        _, (hidden, _) = lstm(lstm_input.unsqueeze(1))
        return hidden.squeeze(0)

class AttentionAggregator(nn.Module):
    """注意力聚合器 - 区分不同邻居的重要性"""
    def forward(self, features, neighbors, self_node):
        # 计算每个邻居的注意力权重
        neighbor_hidden = torch.stack([features[neighbor] for neighbor in neighbors])
        self_hidden = features[self_node].expand_as(neighbor_hidden)
        
        # 拼接后通过注意力网络
        combined = torch.cat([self_hidden, neighbor_hidden], dim=-1)
        attention = torch.softmax(self.attention_net(combined), dim=0)
        
        return torch.sum(attention * neighbor_hidden, dim=0)
```

**3. 特征融合**：将节点自身特征与聚合的邻居特征融合

$$h_v^{(k)} = \sigma\left(W^{(k)} \cdot \text{CONCAT}\left(h_v^{(k-1)}, h_{\mathcal{N}(v)}^{(k)}\right)\right)$$

或者使用拼接后的门控机制：

$$h_v^{(k)} = (1 - \epsilon) \cdot h_v^{(k-1)} + \epsilon \cdot h_{\mathcal{N}(v)}^{(k)}$$

### 完整的前向传播

```python
def forward(self, nodes, num_samples_list=[10, 10]):
    """
    nodes: 要计算的节点列表
    num_samples_list: 每层采样的邻居数量 [k1, k2, ...]
    """
    # 存储每层的节点表示
    hidden = {node: self.embedding(node) for node in nodes}
    
    for k in range(len(num_samples_list)):
        # 采样邻居
        sampled_neighbors = self.sample_neighbors(nodes, num_samples_list[k])
        # 聚合邻居信息
        aggregated = [self.aggregators[k](hidden, neighbors) 
                      for node, neighbors in zip(nodes, sampled_neighbors)]
        # 更新节点表示
        hidden = {nodes[i]: self.update(hidden[nodes[i]], aggregated[i]) 
                  for i in range(len(nodes))}
    
    return hidden
```

| 特性 | 说明 |
|------|------|
| **归纳学习** | 可泛化到未见节点 |
| **采样效率** | 固定采样数量，支持大规模图 |
| **灵活聚合** | 支持多种聚合函数 |
| **Mini-batch | 无需整图参与训练 |

## 图注意力网络 (GAT)

**图注意力网络**（Graph Attention Network，GAT）由 Petar Veličković 等人于 2018 年提出，通过引入注意力机制，让模型自动学习不同邻居的重要性。

### 注意力机制原理

GAT 的核心是**自注意力**（Self-Attention）：每个节点会为每个邻居分配一个注意力系数，这个系数决定了该邻居信息的重要性。

$$\alpha_{ij} = \frac{\exp\left(\text{LeakyReLU}\left(a^T [W h_i \| W h_j]\right)\right)}{\sum_{k \in \mathcal{N}(i)} \exp\left(\text{LeakyReLU}\left(a^T [W h_i \| W h_k]\right)\right)}$$

其中：
- $W$：线性变换矩阵
- $a$：注意力向量
- $[\cdot \| \cdot]$：拼接操作
- $\mathcal{N}(i)$：节点 $i$ 的邻居集合（包括自身）

### 多头注意力

为了稳定学习过程并增强模型容量，GAT 采用**多头注意力**机制：

$$h_i' = \underbrace{\|_{k=1}^{K} \sigma\left(\sum_{j \in \mathcal{N}(i)} \alpha_{ij}^k W^k h_j\right)}_{\text{K 个独立的注意力头}}$$

- 最后一层：拼接各头输出
- 中间层：平均各头输出

### 代码实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class GraphAttentionLayer(nn.Module):
    """图注意力层"""
    def __init__(self, in_features, out_features, dropout=0.1, alpha=0.2, concat=True):
        super(GraphAttentionLayer, self).__init__()
        self.in_features = in_features
        self.out_features = out_features
        self.dropout = dropout
        self.alpha = alpha
        self.concat = concat

        # 线性变换矩阵 W
        self.W = nn.Parameter(torch.zeros(size=(in_features, out_features)))
        nn.init.xavier_uniform_(self.W.data, gain=1.414)
        
        # 注意力向量 a
        self.a = nn.Parameter(torch.zeros(size=(2 * out_features, 1)))
        nn.init.xavier_uniform_(self.a.data, gain=1.414)
        
        self.leakyrelu = nn.LeakyReLU(self.alpha)

    def forward(self, input, adj):
        # input: [N, in_features]
        # adj: 邻接矩阵 [N, N]，有连接为1，无连接为0
        
        h = torch.mm(input, self.W)  # [N, out_features]
        N = h.size()[0]
        
        # 计算注意力系数
        a_input = torch.cat([h.repeat(1, N).view(N * N, -1), 
                              h.repeat(N, 1)], dim=1).view(N, -1, 2 * self.out_features)
        # a_input: [N, N, 2*out_features]
        
        e = self.leakyrelu(torch.matmul(a_input, self.a).squeeze(2))
        # e: [N, N]
        
        # 处理无连接节点（mask）
        zero_vec = -9e15 * torch.ones_like(e)
        attention = torch.where(adj > 0, e, zero_vec)
        attention = F.softmax(attention, dim=1)
        attention = F.dropout(attention, self.dropout, training=self.training)
        
        # 加权聚合
        h_prime = torch.matmul(attention, h)
        
        if self.concat:
            return F.elu(h_prime)
        else:
            return h_prime


class GAT(nn.Module):
    """多层图注意力网络"""
    def __init__(self, n_features, n_hidden, n_classes, n_heads=8, dropout=0.1, alpha=0.2):
        super(GAT, self).__init__()
        self.dropout = dropout
        
        # 多头注意力层
        self.att1 = [GraphAttentionLayer(n_features, n_hidden, dropout, alpha, concat=True) 
                     for _ in range(n_heads)]
        for i, attention in enumerate(self.att1):
            self.add_module('attention1_{}'.format(i), attention)
        
        self.att2 = [GraphAttentionLayer(n_hidden, n_classes, dropout, alpha, concat=True) 
                     for _ in range(n_heads // 2)]
        for i, attention in enumerate(self.att2):
            self.add_module('attention2_{}'.format(i), attention)

    def forward(self, x, adj):
        # 第一层多头注意力
        x = torch.cat([att(x, adj) for att in self.att1], dim=1)
        x = F.dropout(x, self.dropout, training=self.training)
        
        # 第二层多头注意力
        x = torch.cat([att(x, adj) for att in self.att2], dim=1)
        return F.log_softmax(x, dim=1)
```

### GAT vs GCN

| 特性 | GCN | GAT |
|------|-----|-----|
| **邻居重要性** | 固定（基于度归一化） | 可学习（注意力机制） |
| **计算复杂度** | $O(N)$ | $O(N^2)$（全注意力） |
| **并行化能力** | 高度可并行 | 注意力可并行 |
| **适用场景** | 同构图、规则结构 | 异构图、不规则关系 |

## 消息传递神经网络 (MPNN)

**消息传递神经网络**（Message Passing Neural Network，MPNN）由 Justin Gilmer 等人于 2017 年提出，是一个统一的框架，将大多数图神经网络归纳为两个阶段的迭代。

### 统一框架

MPNN 将 GNN 的运作机制抽象为消息传递的两个阶段：

**1. 消息传递阶段**（$T$ 轮）：

$$m_v^{(t+1)} = \sum_{w \in \mathcal{N}(v)} M^{(t)}(h_v^{(t)}, h_w^{(t)}, e_{vw})$$

$$h_v^{(t+1)} = U^{(t)}(h_v^{(t)}, m_v^{(t+1)})$$

**2. 读出阶段**：

$$\hat{y} = R(\{h_v^{(T)} | v \in G\})$$

其中：
- $M^{(t)}$：消息函数
- $U^{(t)}$：更新函数
- $R$：读出函数（将节点表示聚合成图表示）
- $e_{vw}$：边特征

### 常见实例化

| 模型 | 消息函数 $M$ | 更新函数 $U$ | 读出函数 $R$ |
|------|--------------|--------------|--------------|
| **GCN** | $h_w \cdot \frac{1}{\sqrt{d_v d_w}}$ | $\sigma(W \cdot \text{CONCAT}(h_v, m_v))$ | Mean/Sum pooling |
| **GAT** | $\text{Attention}(h_v, h_w) \cdot W h_w$ | $\sigma(W \cdot \text{CONCAT}(h_v, m_v))$ | Mean pooling |
| **GraphSAGE** | $AGG(\{h_w\})$ | $\sigma(W \cdot \text{CONCAT}(h_v, m_v))$ | Mean/LSTM pooling |
| **Edge Net** | $M_e(h_v, h_w, e_{vw})$ | $U(h_v, \sum m_e)$ | Pooling |

### 代码实现

```python
class MessagePassingLayer(nn.Module):
    """通用消息传递层"""
    def __init__(self, node_dim, edge_dim, hidden_dim, message_steps=3):
        super().__init__()
        self.message_steps = message_steps
        
        # 消息函数
        self.message_net = nn.Sequential(
            nn.Linear(2 * node_dim + edge_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )
        
        # 更新函数（GRU门控）
        self.update_net = nn.GRU(hidden_dim, node_dim, batch_first=True)
        
        # 读出函数
        self.readout_net = nn.Sequential(
            nn.Linear(node_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)
        )

    def message(self, h_v, h_w, e_vw):
        """生成消息"""
        # h_v: [E, node_dim] - 中心节点特征
        # h_w: [E, node_dim] - 邻居节点特征  
        # e_vw: [E, edge_dim] - 边特征
        return self.message_net(torch.cat([h_v, h_w, e_vw], dim=-1))

    def aggregate(self, messages, node_idx, num_nodes):
        """聚合消息"""
        aggregated = torch.zeros(num_nodes, messages.size(-1), device=messages.device)
        aggregated = aggregated.scatter_add(0, node_idx.unsqueeze(-1).expand_as(messages), messages)
        return aggregated

    def update(self, h_old, m_new):
        """更新节点表示"""
        # m_new: [num_nodes, hidden_dim]
        # h_old: [num_nodes, node_dim]
        combined = torch.cat([h_old, m_new], dim=-1)
        # 简化的更新
        return torch.sigmoid(combined[:, :h_old.size(-1)]) * h_old + \
               torch.sigmoid(combined[:, h_old.size(-1):]) * m_new

    def forward(self, h, edge_index, edge_attr):
        # edge_index: [2, E] - 边索引
        # edge_attr: [E, edge_dim] - 边特征
        
        src, dst = edge_index
        
        for _ in range(self.message_steps):
            # 1. 生成消息
            messages = self.message(h[dst], h[src], edge_attr)
            
            # 2. 聚合消息
            m = self.aggregate(messages, dst, h.size(0))
            
            # 3. 更新节点表示
            h = self.update(h, m)
        
        return h

    def readout(self, h, batch_idx):
        """图级别读出"""
        # 按图分组聚合
        graph_representations = []
        for g in range(batch_idx.max().item() + 1):
            mask = batch_idx == g
            graph_h = h[mask]
            # 使用注意力权重的加权平均
            weights = torch.softmax(self.readout_net(graph_h), dim=0)
            graph_repr = (weights * graph_h).sum(dim=0)
            graph_representations.append(graph_repr)
        return torch.stack(graph_representations)
```

## 图同构网络 (GIN)

**图同构网络**（Graph Isomorphism Network，GIN）由 Keyulu Xu 等人于 2019 年提出，是第一个在理论上证明具有区分任意图结构能力的 GNN 架构。

### 理论背景：Weisfeiler-Lehman 算法

GIN 的设计灵感来自图论中的 **Weisfeiler-Lehman (WL) 同构测试**。WL 算法通过迭代地聚合邻居标签来区分图的同构性：

```
初始：每个节点有自己的初始颜色
迭代：
  1. 每个节点收集邻居的颜色（多集合）
  2. 通过哈希函数生成新的颜色
  3. 如果两次迭代后颜色分布不同，图必不同构
```

### GIN 的设计原则

GIN 遵循两个关键原则：

**原则 1**：邻居聚合需要是**单射的**（Injective）

$$h_v^{(k)} = \text{MLP}^{(k)}\left((1 + \epsilon^{(k)}) \cdot h_v^{(k-1)} + \sum_{w \in \mathcal{N}(v)} h_w^{(k-1)}\right)$$

**原则 2**：读出函数需要是**多集上的单射函数**

$$h_G = \text{CONCAT}\left(\text{READOUT}\left(\{h_v^{(k)} | v \in G\}\right) \text{ for } k = 1, ..., K\right)$$

### 为什么 GIN 更强大？

| 聚合方式 | 表达能力 | 能否区分 |
|----------|----------|----------|
| **Mean** (GCN) | 有限 | 不能区分邻居分布不同但均值相同的情况 |
| **Max** (GraphSAGE) | 有限 | 不能区分多集的大小 |
| **Sum** (GIN) | 最强 | 能够区分任意多集 |

例如：对于聚合 $[0, 1]$ 和 $[0.5, 0.5]$：
- Mean 聚合：两者都得到 0.5（无法区分）
- Sum 聚合：得到 1 和 1（无法区分）
- 但对于 $[0, 1, 2]$ 和 $[1, 1, 1]$：
  - Mean：都得到 1（无法区分）
  - Sum：3 vs 3（无法区分）
  - 需要更深的网络或更好的特征

### 代码实现

```python
class GIN(nn.Module):
    """图同构网络"""
    def __init__(self, n_features, n_hidden, n_classes, n_layers=5, eps=0):
        super(GIN, self).__init__()
        self.n_layers = n_layers
        self.eps = eps
        
        # 输入映射
        self.input_net = nn.Sequential(
            nn.Linear(n_features, n_hidden),
            nn.ReLU(),
            nn.Linear(n_hidden, n_hidden)
        )
        
        # 每层的MLP（用于消息传递）
        self.mlps = nn.ModuleList([
            nn.Sequential(
                nn.Linear(n_hidden, n_hidden),
                nn.ReLU(),
                nn.Linear(n_hidden, n_hidden)
            )
            for _ in range(n_layers)
        ])
        
        # 可学习的epsilon参数
        self.eps = nn.ParameterList([nn.Parameter(torch.tensor(eps)) 
                                     for _ in range(n_layers)])
        
        # 读出层
        self.readout = nn.Sequential(
            nn.Linear(n_hidden * (n_layers + 1), n_hidden),
            nn.ReLU(),
            nn.Linear(n_hidden, n_classes)
        )

    def forward(self, x, edge_index, batch):
        # x: [N, n_features] - 节点特征
        # edge_index: [2, E] - 边索引
        
        src, dst = edge_index
        
        # 初始化节点嵌入
        h = self.input_net(x)
        hidden_states = [h]  # 用于读出
        
        # 多层消息传递
        for i in range(self.n_layers):
            # 聚合邻居信息
            neighbor_agg = torch.zeros_like(h)
            neighbor_agg = neighbor_agg.scatter_add(0, src.repeat(h.size(-1), 1).T.long(), h[dst])
            
            # 可学习的线性组合
            h_new = (1 + self.eps[i]) * h + neighbor_agg
            
            # MLP更新
            h_new = self.mlps[i](h_new)
            h = h_new
            hidden_states.append(h)
        
        # 读出：将所有层的节点表示拼接
        # 按图聚合
        graph_reprs = []
        for b in range(batch.max().item() + 1):
            mask = batch == b
            # 收集该图中所有层的节点表示
            layer_concat = torch.cat([hs[mask] for hs in hidden_states], dim=-1)
            # Sum pooling
            graph_repr = layer_concat.sum(dim=0)
            graph_reprs.append(graph_repr)
        
        graph_reprs = torch.stack(graph_reprs)
        return self.readout(graph_reprs)
```

### 构建 GNN 模型

```python
import torch
import torch.nn.functional as F
from torch_geometric.nn import GCNConv, GATConv, SAGEConv

class GNNClassifier(torch.nn.Module):
    """通用的 GNN 分类器"""
    def __init__(self, in_channels, hidden_channels, out_channels, num_layers=3, 
                 gnn_type='gcn', dropout=0.5):
        super().__init__()
        self.dropout = dropout
        
        self.convs = torch.nn.ModuleList()
        self.bns = torch.nn.ModuleList()
        
        # 输入层
        if gnn_type == 'gcn':
            self.convs.append(GCNConv(in_channels, hidden_channels))
        elif gnn_type == 'gat':
            self.convs.append(GATConv(in_channels, hidden_channels))
        elif gnn_type == 'sage':
            self.convs.append(SAGEConv(in_channels, hidden_channels))
        self.bns.append(torch.nn.BatchNorm1d(hidden_channels))
        
        # 隐藏层
        for _ in range(num_layers - 2):
            if gnn_type == 'gcn':
                self.convs.append(GCNConv(hidden_channels, hidden_channels))
            elif gnn_type == 'gat':
                self.convs.append(GATConv(hidden_channels, hidden_channels))
            elif gnn_type == 'sage':
                self.convs.append(SAGEConv(hidden_channels, hidden_channels))
            self.bns.append(torch.nn.BatchNorm1d(hidden_channels))
        
        # 输出层
        self.classifier = torch.nn.Linear(hidden_channels, out_channels)

    def forward(self, x, edge_index):
        for i, conv in enumerate(self.convs[:-1]):
            x = conv(x, edge_index)
            x = self.bns[i](x)
            x = F.relu(x)
            x = F.dropout(x, p=self.dropout, training=self.training)
        
        # 最后一层不做激活和dropout
        x = self.convs[-1](x, edge_index)
        x = self.bns[-1](x)
        
        # 读出：平均池化
        x = torch.zeros(x.size(0), x.size(1), device=x.device) if False else None
        
        return self.classifier(x)
```

### 训练与评估

```python
def train(model, data, optimizer, criterion):
    model.train()
    optimizer.zero_grad()
    
    out = model(data.x, data.edge_index)
    loss = criterion(out[data.train_mask], data.y[data.train_mask])
    
    loss.backward()
    optimizer.step()
    
    return loss.item()

def evaluate(model, data):
    model.eval()
    with torch.no_grad():
        out = model(data.x, data.edge_index)
        pred = out.argmax(dim=1)
        
    correct = (pred[data.test_mask] == data.y[data.test_mask]).sum()
    acc = int(correct) / int(data.test_mask.sum())
    return acc

# 主训练循环
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model = GNNClassifier(
    in_channels=dataset.num_node_features,
    hidden_channels=256,
    out_channels=dataset.num_classes,
    gnn_type='gcn'
).to(device)

data = data.to(device)
optimizer = torch.optim.Adam(model.parameters(), lr=0.01, weight_decay=5e-4)
criterion = torch.nn.CrossEntropyLoss()

best_acc = 0
for epoch in range(1, 501):
    loss = train(model, data, optimizer, criterion)
    
    if epoch % 10 == 0:
        acc = evaluate(model, data)
        if acc > best_acc:
            best_acc = acc
        print(f'Epoch: {epoch:03d}, Loss: {loss:.4f}, Test Acc: {acc:.4f}, Best: {best_acc:.4f}')
```

### 使用更高级的模型

```python
from torch_geometric.nn import GENConv, DeepGCNLayer, MLP

class DeepGNN(torch.nn.Module):
    """深层 GNN（使用更先进的消息传递机制）"""
    def __init__(self, in_channels, hidden_channels, out_channels, num_layers=7):
        super().__init__()
        
        self.input_lin = torch.nn.Linear(in_channels, hidden_channels)
        
        # 使用 DeepGCNLayer 支持深层网络
        self.layers = torch.nn.ModuleList()
        for i in range(1, num_layers + 1):
            conv = GENConv(hidden_channels, hidden_channels, aggr='max')
            norm = torch.nn.BatchNorm1d(hidden_channels)
            act = torch.nn.ReLU(inplace=True)
            
            layer = DeepGCNLayer(conv, norm, act, block='res+', dropout=0.1)
            self.layers.append(layer)
        
        self.output_lin = torch.nn.Linear(hidden_channels, out_channels)

    def forward(self, x, edge_index):
        x = self.input_lin(x)
        x = x.tanh()
        
        for layer in self.layers:
            x = layer(x, edge_index)
        
        return self.output_lin(x)
```

# 处理边的原子序数特征
class MoleculeGNN(torch.nn.Module):
    def __init__(self, num_features, hidden_channels, num_classes):
        super().__init__()
        self.conv1 = GCNConv(num_features, hidden_channels)
        self.conv2 = GCNConv(hidden_channels, hidden_channels)
        self.lin = torch.nn.Linear(hidden_channels, num_classes)

    def forward(self, x, edge_index, edge_attr, batch):
        x = F.relu(self.conv1(x, edge_index))
        x = F.relu(self.conv2(x, edge_index))
        
        # 图级别读出
        x = global_mean_pool(x, batch)
        
        return self.lin(x)

### 链接预测

**任务**：预测图中缺失的边

**方法**：
1. 编码两个节点的表示
2. 使用评分函数预测链接存在概率

```python
class LinkPredictor(torch.nn.Module):
    def forward(self, h, edge_index):
        src, dst = edge_index
        # 点积作为评分
        return (h[src] * h[dst]).sum(dim=-1)
    
    def predict_link(self, h1, h2):
        # 预测任意两节点间的链接
        return (h1 * h2).sum(dim=-1)
```

### 图生成

**任务**：生成符合某种分布的新图

**应用**：
- 药物分子设计
- 代码生成
- 场景图生成

```python
from torch_geometric.nn import ARMAConv

class GraphGenerator(torch.nn.Module):
    """基于自编码器的图生成模型"""
    def __init__(self, in_channels, hidden_channels):
        super().__init__()
        # 编码器
        self.encoder = GCNConv(in_channels, hidden_channels)
        # 解码器：预测边
        self.decoder = torch.nn.Sequential(
            torch.nn.Linear(2 * hidden_channels, hidden_channels),
            torch.nn.ReLU(),
            torch.nn.Linear(hidden_channels, 1),
            torch.nn.Sigmoid()
        )

    def encode(self, x, edge_index):
        h = F.relu(self.encoder(x, edge_index))
        return h

    def decode_all(self, h):
        # 解码所有可能的边
        adj = torch.sigmoid(torch.mm(h, h.t()))
        return adj
```

## GNN 的挑战与前沿方向

### 当前挑战

| 挑战 | 描述 | 研究方向 |
|------|------|----------|
| **过平滑** | 层数加深导致节点表示趋同 | 残差连接、归一化 |
| **过参数化** | 大图上参数量爆炸 | 图采样、知识蒸馏 |
| **异构图** | 不同类型节点/边 | R-GCN, HAN, HGT |
| **动态图** | 图结构随时间变化 | DGN, TGN |
| **大规模图** | 亿级节点的高效处理 | Cluster-GCN, GraphSAINT |
| **理论解释** | 理解 GNN 的表达能力 | WL测试、表达能力分析 |

### 前沿方向

**1. Transformer for Graph**

将 Transformer 的自注意力机制应用于图结构数据：

```python
from torch_geometric.nn import TransformerConv

class GraphTransformer(nn.Module):
    def __init__(self, in_channels, hidden_channels, num_heads=4, num_layers=3):
        super().__init__()
        self.layers = nn.ModuleList([
            TransformerConv(in_channels if i == 0 else hidden_channels,
                          hidden_channels, heads=num_heads, concat=True)
            for i in range(num_layers)
        ])
        self.norms = nn.ModuleList([nn.LayerNorm(hidden_channels * num_heads) 
                                     for _ in range(num_layers)])

    def forward(self, x, edge_index):
        for layer, norm in zip(self.layers, self.norms):
            x = layer(x, edge_index)
            x = norm(x)
            x = F.dropout(x, p=0.1, training=self.training)
        return x
```

**2. 异构图神经网络**

```python
from torch_geometric.nn import RGCNConv, HANConv

class HeterogeneousGNN(nn.Module):
    def __init__(self, node_types, edge_types, hidden_channels):
        super().__init__()
        # 关系图卷积
        self.rgcn1 = RGCNConv(hidden_channels, hidden_channels, num_relations=len(edge_types))
        # 语义注意力
        self.han = HANConv(hidden_channels, hidden_channels, metadata=(node_types, edge_types))

    def forward(self, x_dict, edge_index_dict):
        # 异构图前向传播
        out = self.han(x_dict, edge_index_dict)
        return out
```
